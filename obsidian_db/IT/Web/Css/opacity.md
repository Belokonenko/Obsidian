- прозрачности элемента.  
- принимает значения от 0 до 1 
- влияет на все потомки элемента
- эл активен
```css

`/* Полностью прозрачный */ 
.element1 {   opacity: 0; }  
/* Частично прозрачный */ 
.element2 {   opacity: 0.5; }  
/* Полностью непрозрачный (по умолчанию) */ 
.element3 {   opacity: 1; }`

```
